# Luacuit 多旋翼飞行控制器用户手册

## 概述

本 Luacuit 脚本实现了一个功能完整的多旋翼飞行控制器，适用于 Cimulink/Synaxis 仿真环境。控制器基于**刚体动力学模型**，支持 **4 种飞行模式**（定点、姿态、增稳、手动）以及**视线跟踪模式**。核心算法包括：

- 基于四元数的姿态控制（无欧拉角奇异）
- 动态力矩分配（支持非对称机架）
- 完整的角速度闭环（PID + 前馈 + 科里奥利补偿）
- 动态限幅（基于物理能力自动调整）
- 基于电机物理参数的精确混控

---

## 功能特性

| 特性 | 说明 |
|------|------|
| **坐标系** | 世界：东-天-南；机体：左-上-前 |
| **控制频率** | 可调（默认 40 Hz） |
| **电机数量** | 4 个（可配置位置、推力系数、旋转方向） |
| **力矩分配** | 基于 4x4 线性方程组求解基础推力 + 伪逆增量 |
| **角速度控制** | PID + 运动学前馈 + 科里奥利前馈 |
| **动态限幅** | 电机转速、积分项、角加速度、速度误差等自动适配质量/惯量 |
| **高度控制** | 垂直速度 PID + 倾斜保护 + 倒飞逻辑 |
| **水平控制** | 世界坐标系水平速度 PI + 加速度 → 姿态映射 |
| **视线跟踪** | 四元数直接对齐机头与视线向量，b11 一键回平 |

---

## 飞行模式

| 模式 | 名称 | 说明 | 切换按键 |
|------|------|------|-----------|
| `MODE_POSITION` | 定点模式 | 外环水平速度控制，内环角速度闭环；油门为垂直速度指令 | b6 |
| `MODE_STAB` | 姿态模式 | 摇杆直接控制机体倾斜方向（重力向量控制） | b7 |
| `MODE_ACRO_PLUS` | 增稳模式 | 摇杆输入为期望姿态角（增量四元数），内环角速度闭环 | b8 |
| `MODE_ACRO` | 手动模式 | 摇杆直接控制角速度（无角度限制） | b9 |
| `MODE_VIEW` | 视线跟踪 | 机头自动指向视线目标，滚转手动，油门为垂直速度 | b10 |

> 按键 `b4` 使所有电机推力反向（倒飞）  
> 按键 `b11` 在视线模式中激活一键回平（重力向量对齐）

## 参数配置

所有参数均在 `define()` 函数中声明，可通过 Luacuit UI 实时调整。以下按功能分类列出关键参数：

### 基础参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `HZ` | 40 | 控制循环频率 (Hz) |
| `g_eng` | 600 | 总推力缩放系数（与油门曲线相乘） |
| `throttle_curve` | 2.5 | 油门曲线指数（>1 增加低油门灵敏度） |
| `eng_clamp` | 750 | 电机转速指令绝对值限幅 |

### 电机物理参数
| 参数 | 默认值 | 单位 | 说明 |
|------|--------|------|------|
| `motor_x1` ~ `motor_x4` | ±2, -2, -2, 2 | 米 | 电机 X 坐标（左为正） |
| `motor_z1` ~ `motor_z4` | 2, 2, -2, -2 | 米 | 电机 Z 坐标（前为正） |
| `motor_y1` ~ `motor_y4` | 1.0 | 米 | 电机 Y 坐标（上为正） |
| `yaw_sign_1` ~ `yaw_sign_4` | -1, 1, -1, 1 | 无 | 电机旋转方向对偏航力矩的贡献 |
| `thrust_coeff_1` ~ `thrust_coeff_4` | -5, 5, -5, 5 | N / 转速 | 推力系数（符号表示推力方向） |
| `torque_coeff` | 5 | Nm / 转速 | 反扭力矩系数 |

### 手动模式 (Acro) 参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `stick_gain_roll` | 1.0 | 滚转摇杆增益 |
| `stick_gain_pitch` | -1.4 | 俯仰摇杆增益 |
| `stick_gain_yaw` | 0.7 | 偏航摇杆增益 |
| `man_rate_p_roll/pitch/yaw` | 5.0 | 角速度环 P 增益 |
| `man_rate_i_roll/pitch/yaw` | 0.5, 0.5, 0.2 | 角速度环 I 增益 |
| `i_limit` | 2.0 | 角速度积分限幅 (rad/s) |
| `max_roll_rate` / `pitch` / `yaw` | 10.0 | 最大期望角速度 (rad/s) |
| `man_ff_enable` | 1 | 运动学前馈使能 |
| `ff_gyro_enable` | 1 | 科里奥利前馈使能 |
| `ff_gyro_scale` | 0.2 | 科里奥利前馈强度 |

### 定点/姿态模式参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `att_p` | 1.5 | 角度环 P 增益 |
| `max_angle` | 40 | 最大倾斜角度 (度) |
| `max_horiz_speed` | 20.0 | 最大水平速度 (m/s) |
| `max_vert_speed` | 20.0 | 最大垂直速度 (m/s) |
| `vert_p` | 1 | 垂直速度 P 增益 |
| `vert_d` | 0.2 | 垂直速度 D 增益 |
| `vert_i` | 0.5 | 垂直速度 I 增益 |
| `horiz_p` | 3 | 水平速度 P 增益 |
| `horiz_d` | 0.01 | 水平速度 D 增益 |
| `horiz_i` | 0.25 | 水平速度 I 增益 |
| `horiz_i_limit` | 5.0 | 水平速度积分限幅 (m/s 累积) |
| `angle_i_pitch/roll` | 0.5 | 姿态角积分增益 |
| `yaw_lpf` | 0.2 | 偏航通道低通滤波系数 |

### 视线模式参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `k_pitch` (硬编码) | 2.0 | 俯仰跟踪比例增益 |
| `k_yaw` (硬编码) | 2.0 | 偏航跟踪比例增益 |
| `k_level` (硬编码) | 2.0 | 回平过程角速度增益 |

> 视线模式需从总线 `fpvc.abs_forward` 读取目标视线向量（世界坐标系）。回平功能通过 b11 按键触发，基于重力向量对齐。

## 使用方法

### 1. 在 Luacuit 编辑器中加载脚本
将提供的 `PX4__25.lua` 复制到 Luacuit 编辑器，点击 `Apply` 编译。

### 2. 配置电机参数
根据实际机架尺寸和电机特性，在 UI 中修改以下参数：
- 四个电机的坐标 (`motor_x1` … `motor_z4`)
- 推力系数 (`thrust_coeff_1` … `thrust_coeff_4`)
- 偏航符号 (`yaw_sign_1` … `yaw_sign_4`)
- 反扭力矩系数 (`torque_coeff`)

### 3. 连接遥控器
确保总线 `fpvin` 包含摇杆输入：
- `ly` : 油门/垂直速度（-1~1）
- `rx` : 滚转
- `ry` : 俯仰
- `lx` : 偏航
- `b4` : 倒飞开关
- `b6` … `b11` : 模式切换与功能按键

### 4. 选择飞行模式
按下对应按键即可切换模式，当前模式会在 `debug_attitude_mode` 输出端口显示（布尔值仅指示定点模式）。

### 5. 调试输出
脚本提供以下输出端口，可用于观察：
- `debug_ctrl_roll/pitch/yaw` : 内环控制量
- `debug_eng` : 总推力指令
- `debug_attitude_mode` : 是否为定点模式

## 注意事项

1. **推力系数符号**：脚本中 `thrust_coeff` 的符号应保证电机正转速时推力向上。若实际电机安装反了，需修改系数符号或旋转方向。
2. **质心偏移**：控制器会自动读取 `phys.positionCenter()` 计算机体力臂，因此加载后无需手动调整。但需确保电路板位置 (`phys.position()`) 正确。
3. **动态限幅**：依赖 `max_thrust_ratio`（硬编码为 2.0）和 `dyn_limit_ratio`（未在 define 中，但可扩展），当前版本未暴露。若需动态调整，可在 define 中添加相应参数。
4. **视线模式**：需要外部提供 `fpvc.abs_forward` 向量，否则默认指向 (0,0,1)。
5. **倒飞**：按下 b4 后总推力方向取反，可全油门倒飞。但注意高度控制逻辑中倾斜角超过 90° 时会自动取反垂直速度控制。
6. **性能**：脚本包含矩阵求逆（4x4 和 3x3），每帧执行一次，在 40 Hz 下开销可接受。

## 文件结构

```
PX4__25.lua
├── 坐标系与模式定义
├── 状态变量（积分、滤波、历史）
├── 辅助函数（四元数、欧拉角、旋转矩阵等）
├── 可调参数定义（define()）
├── 线性方程组求解（solve_linear_4x4）
├── 核心角速度控制函数（angular_rate_control）
├── 垂直/水平速度控制（horiz_stable）
└── 主循环（loop()）
```

## 依赖与兼容性

- **运行环境**：Luacuit（Cimulink/Synaxis 内置）
- **物理引擎**：需支持 `phys.mass()`、`phys.velocity()`、`phys.angularVelocity()`、`phys.inertiaTensor()`、`phys.positionCenter()`、`phys.position()`、`phys.quaternionToWorld()` 等接口。
- **总线**：需要 `fpvin` 和 `fpvc` 总线提供输入/输出。
